Methods

initialize()

framework/ITundraPlugin.js, line 84
Called when plugin is loaded to TundraClient. Implementation can override this. Note: this.framework is set at this stage.

pluginPropertyName(){String}

framework/ITundraPlugin.js, line 75
Called to resolve property name for Tundra.plugins. registration. Implementations can override this, if not overridden will return plugin name. The returned name will be lowercased by Tundra, eg. MyPlugin > myPlugin, Voip > voip.
Returns:
name of the plugin property

postInitialize()

framework/ITundraPlugin.js, line 99
Called after all plugins have been loaded to TundraClient. Useful if your plugin needs to fetch and use other plugins. Implementation can override this.

uninitialize()

framework/ITundraPlugin.js, line 115
Called when plugin is unloaded from TundraClient. Implementation can override this. Note: this.framework is still valid at this stage.
